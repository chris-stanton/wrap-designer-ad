# Lamborghini Aventador Parallax Website

---

## Technologies Used
  1. HTML5
  2. CSS3
  3. ES6
  4. Flex Box

---

## Install App
  1. Clone or Download zip file from repo
  2. Open index.html file in browser

---

### Description:
###### An HTML and CSS remake of an Lamborghini Aventador.
